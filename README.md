# Octopy Web Console
Access the console through the web

#### REQUIREMENTS
- font iosevka - [installation](https://github.com/be5invis/Iosevka)
- php 5.6 or later with `proc_open` function
- web server like apache2 or nginx

#### INSTALLATION
just put to your web server
